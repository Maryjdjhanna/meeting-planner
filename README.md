# Meeting Planner

Use `meeting-planner.md` as a template for planning each meeting.

## Meeting Planner Web App

Open `index.html` in your browser to use the interactive planner for cross-region scheduling.

Features:
- Region group + region dropdown list for each attendee (AMER, EMEA, APAC)
- Add/remove attendees
- Shared working-hours window
- Suggested overlapping slots across regions with local times

Project files:
- `index.html` (app structure)
- `styles.css` (styling)
- `app.js` (planner logic)

Quick open on macOS:
`open index.html`

Quick start:
1. Copy the template for a new meeting note.
2. Fill in each section before the meeting.
3. Update decisions and action items during or after the meeting.

Example:
`cp meeting-planner.md meeting-2026-03-05.md`
